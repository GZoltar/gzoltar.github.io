The GZoltar toolset

Automatic Testing & Debugging using Spectrum-based Fault Localization (SFL)

GZoltar is a framework for automating the testing and debugging phases of the software development life-cycle. At the moment, the framework is provided as a command line interface, ant task, maven plug-in, and as an Eclipse plugin; and it integrates seamlessly with JUnit tests.
GZoltar is an on-going project. Hence, there might be interesting features that have not yet been implemented, or even unexpected behavior. We would like to hear from you in case you foresee an unimplemented feature or if you observe unexpected behavior. Send us an email to mail [at] gzoltar [dot] com and help us to improve GZoltar.
